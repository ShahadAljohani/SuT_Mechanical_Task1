## About this task
Torque Calculations for a Robotic Arm Joint

----------
 ### Equations:

- **Gear Ratio:**
  
  $$\text{Gear Ratio} = \frac{\text{Driven (Output Torque)}}{\text{Driver (Input Torque)}}$$

- **Torque Equation:**
  
  $$\vec{\tau} = \vec{r} \times \vec{F}$$

#### Questions: 


#### 1| Calculate Torque:
Determine the required torque for each motor in the robotic arm and specify which motor is best suited for each joint.

- torque of 1 kg:

J1 = 1kg * 4cm = 4kg/cm

J2 = 1kg * 10cm = 10kg/cm + 4kg/cm = 14 kg/cm

J3 = 1kg * 15 cm = 15kg/cm + 14kg/cm = 29kg/cm

- motors for each joint:

J1 = AGFRC TRX4M

J2 = Mitec HS-7954SH

J3 = Traxxas 2255





#### 2|Can the previous motors, specified to handle 1 kg, be adapted to lift 2 kg using gears?

- What potential drawbacks might occur when using gears to increase load capacity?

- What alternative solutions could be considered?

------

Yes, by adding gears since gears increases the torque.

Drawbacks:
- increasing the dear ratio 4x means the speed will be decreased 4x
  
- range of motion my change
  
- more gears may cause more friction which will reduce efficiency
  
- gears will add weight and size

to compensate the drawbacks:
- selecting motor with higher torque (by 2x)
  
- reduce load's weight
  
- upgrade the power supply with proper voltage and current to help maintain speed
